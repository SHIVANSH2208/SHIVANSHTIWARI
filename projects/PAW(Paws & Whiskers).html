PAW(Paws and Whiskers)

  • Tech Stack: HTML5, CSS, JavaScript, Tailwind CSS
  • Project Link: P.A.W
  • Certificate URL: HackIndia

Developed Key Features: Built functionalities for Pet Adoption and Pet Donation, enabling users to adopt or donate pets seamlessly.

Implemented Emergency Care: Integrated an Emergency Care feature to connect users with nearby pet care services.

Enhanced User Engagement: Created sections for Pet News and Blogs to provide the latest updates and stories in the pet community.

Improved User Interaction: Added Volunteer, Sign-Up, Contact Page, and Newsletter Subscription to boost engagement and support.

Optimized User Experience: Enabled Dark/Light Mode Toggle and Multi-language Support with Google Translate for accessibility.

Data Management: Utilized a database to store user data for auto-fill, improving efficiency and user experience.
